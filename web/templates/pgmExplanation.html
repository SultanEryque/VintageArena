Description

C++ is an upgrade of the C programming language. It is considered a low level language. low-level language is described as one that is close to machines, machine language. Its an upgrade because it also employs OOP which is not available in C. The language is platform independent hence can compiled in any OS . Ulla Kirch-Prinz and Peter Prinz book A Complete Guide To Programming In C++ states C++ was created by Bjarne Stroustrap and his team at Bell Labs to help implement simulation projects in an object oriented and efficient way.
The sample code below is a simple program that basically takes two numbers from user input i.e a and b and adds them to provide the sum e.g if a user inputs 50 and 30. It will output The sum is 80. How Cool!!!. 


//The program adds two user inputs and outputs the sum
#include        //Declaration of cout, cin
#include          //String manipulation
using namespace std;

int main()
{
   int a, b, sum;         //variable declaration
   string p "Welcome, \nKindly input two integers to add";

   cout << p << endl;
   
   cin >> a;       //reads 1st int and assigns to var a
   cin >> b;       //reads 2nd int and assigns to var b

   sum = a + b;    //results assigned to variable

   cout << "The sum is " << sum << endl;

   return 0;
}
